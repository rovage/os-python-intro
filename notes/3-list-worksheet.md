Pattern to follow:

**list_name[index]** for a single list element

**list_name[start : end : step]** for a sublist

Exercises
---------

numbers = [1, 2, 3, 4, 5, 6, 233]

| Statement  | Result      
|---|------------------------------
| numbers[0]  |           
| numbers[-1] |           |
| numbers[0:2]  |           |
| numbers[1:] |           |
| numbers[:-1]  |           |
| numbers[::2] |           |

Answers:
* [1,3,5,233]
* 1
* [1,2]
* [1,2,3,4,5,6]
* 233
* [2,3,4,5,6,233]
